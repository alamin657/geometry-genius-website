3.What is Event Bubble?

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that
event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.

What is callback function?

Ans: A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.
